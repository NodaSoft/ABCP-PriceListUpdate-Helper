Использование
=============

Необходимое ПО
==============

Перед запуском должны быть установлены:

- Bash (версия 4+)
- curl

Проверка:

	bash --version
	curl --version

Перед запуском задайте переменные окружения с доступом к API:

	export API_USERNAME="..."
	export API_USERPSW="..."
	export API_HOST="api.abcp.ru"

Скрипт принимает 2 или 3 аргумента:

Идентификатор склада

Путь к прайс-листу

Необязательный 3-й аргумент `inc` для инкрементальной загрузки (`fileTypeId=4`).

Пример полной загрузки:

	sh ./update_price.sh 000000 /tmp/pricelist.csv

Пример инкрементальной загрузки:

	sh ./update_price.sh 000000 /tmp/pricelist.csv inc
